# Hospital Patient Records Dashboard Using Tableau


# Project Objective 

* Analyze hospital patient records and admission trends.
* Monitor department-wise patient count and revenue.
* Identify the most common diseases treated in the hospital.
* Compare patient distribution based on gender and admission type.
* Build an interactive Tableau dashboard to support healthcare decision-making.

  
# Dataset Used

* **Dataset Name:** Hospital Data Analysis Dataset.
* Contains patient demographic and hospital admission information.
* Includes patient details, department, disease, doctor, admission date, and revenue.
* Covers admission types, gender, treatment conditions, and monthly admissions.
* Used to analyze hospital performance and patient management through Tableau.
[🏥 Hospital Data Analysis Dataset (.xlsx)](https://github.com/Ezhilarasi-123/Tableau_Miniproject/blob/main/hospital_data_analysis_enhanced.xlsx)

# Business Questions Answered 

* Which hospital department has the highest number of patients?
* Which department generates the highest revenue?
* What are the most frequently diagnosed diseases?
* How do monthly patient admissions change throughout the year?
* What is the distribution of patients by gender and admission type?

# KPI Dashboard & Interaction 

* Displays key KPIs including **Total Patients, Total Revenue, Average Revenue, Readmission Rate, and Total Treatment Cost**.
* Interactive filters for **Department, Gender, Admission Month, Admission Type, Doctor Name, and Disease Condition**.
* Dynamic dashboard updates instantly based on selected filters.
* Visualizes patient distribution, department performance, and admission trends.
* Enables quick analysis for hospital administrators and healthcare professionals.

  ![Hospital Patient Records Dashboard](https://raw.githubusercontent.com/Ezhilarasi-123/Tableau_Miniproject/main/Screenshot%202026-07-08%20114813.png)

# Project Process 

1. Collected and imported the Hospital Data Analysis dataset into Tableau.
2. Cleaned and prepared the data by correcting formats and handling missing values.
3. Created calculated fields and KPIs for patient count, revenue, and performance analysis.
4. Designed interactive charts, KPI cards, filters, and dashboards using Tableau.
5. Developed a professional healthcare dashboard to generate actionable insights.

   ![Hospital Patient Records Dashboard](https://raw.githubusercontent.com/Ezhilarasi-123/Tableau_Miniproject/main/Screenshot%202026-07-08%20114813.png)

# Project Insights 

* **General Medicine** recorded the highest number of patient admissions and generated the highest revenue among all departments.
* The patient population is almost evenly distributed between **male and female** patients.
* Monthly admissions remained relatively stable during the first half of the year before showing a slight decline in later months.
* Common conditions such as **Hypertension, Diabetes, Stroke, Respiratory Disorders, Kidney Stones, and Osteoarthritis** account for a large share of hospital admissions.
* Interactive filtering by department, doctor, admission type, and disease enables detailed analysis of hospital performance and patient care.


# Conclusion

The **Hospital Patient Records Dashboard Using Tableau** successfully transforms hospital data into an interactive and insightful analytics solution. By visualizing patient admissions, departmental performance, disease trends, revenue, and demographic information, the dashboard helps healthcare professionals make informed decisions. This project demonstrates how Tableau can be used to improve hospital operations, optimize resource planning, enhance patient management, and support data-driven healthcare strategies.
